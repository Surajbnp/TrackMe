# TrackMe
TrackMee is a simple task website, where can add new task, edit, delete and update task separately. Responsive and Modern UI designs for better user experiance with smooth backend.

# Functionality
- Add New Task
- Delete Task
- Update Task
- Toggle Status Of Task
